This a supervised Machine Learning approach to determine if a transaction made from an bank account is by the account holder or by a hacker/fraud. This basically uses confusion matrix to identify the precision and recall and plotting them on a graph for analysis. Then we use K-Fold cross validation for supervised calssification of the data.
